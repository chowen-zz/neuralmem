# 架构概述

## 分层架构

```
+-------------------------------------+
|      MCP Server / CLI / SDK         |
+-------------------------------------+
|          NeuralMem 主类              |
+--------+--------+--------+----------+
|存储层   |Embedding|提取层  |检索引擎  |
|SQLite  |FastEmbed|规则+LLM|4策略+RRF |
+--------+--------+--------+----------+
|         知识图谱（NetworkX）          |
+-------------------------------------+
|         生命周期（遗忘曲线）           |
+-------------------------------------+
```

## 关键组件

| 组件 | 实现 | 说明 |
|------|------|------|
| 存储 | SQLite + sqlite-vec | 向量 + 全文 + 关系型 |
| Embedding | FastEmbed ONNX | 本地 384 维向量 |
| 提取 | 规则 + Ollama（可选）| 实体/关系提取 |
| 检索 | 四策略 + RRF | 并行检索融合 |
| 图谱 | NetworkX + SQLite | 实体关系持久化 |
| 生命周期 | Ebbinghaus 曲线 | 遗忘 + 衰减 |
| 消歧 | 规则 + Embedding | 实体去重 |

## 数据流

```
remember(content)
  → 规则提取实体/关系
  → 实体消歧（规则+Embedding）
  → 生成 Embedding 向量
  → 去重检查
  → 写入 SQLite（主表+向量+FTS）
  → 更新知识图谱

recall(query)
  → 并行四策略搜索
  → RRF 融合排序
  → 可选 Cross-Encoder 重排序
  → 返回 SearchResult 列表
```
